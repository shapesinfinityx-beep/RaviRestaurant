# Ravi Restaurant — Premium Website

Authentic Pakistani Cuisine Since 1978 · Al Satwa, Dubai

A premium, scroll-animated restaurant website built with React 19, Vite, Tailwind CSS v4, and Motion (Framer Motion). White + Pakistani green design language with cinematic video backgrounds, parallax effects, magnetic buttons, and word-by-word text reveals.

## Tech Stack

- **React 19** + **TypeScript**
- **Vite 6** (build + dev server)
- **Tailwind CSS v4** (with `@theme` directive)
- **Motion** (scroll animations, parallax, spring physics)
- Compressed video backgrounds (hero + quote section)
- Premium loading screen with progress animation

## Run Locally

**Prerequisites:** Node.js 18+

```bash
npm install
npm run dev
```

Opens at `http://localhost:3000`

## Build

```bash
npm run build
npm run preview
```
